# Reflexão Pessoal sobre o Uso do Git

### O que foi difícil
No começo, o mais difícil foi me adaptar ao padrão de desenvolvimento e aos comandos do git e o que eu deveria fazer em cada etapa.

### O que ficou claro
A importâcia de trabalhar em branches, de ter um padrão claro de desenvolvimento, a importância dos pull requests e dos commits e como é fundamental uma organização do repositório para fazer um projeto.

### O que ainda é confuso
Resolver conflitos ainda é algo que achei um pouco complicado.
