## 📌 Sobre este repositório

Este repositório é o meu portfólio prático de Git, construído na disciplina de Desenvolvimento de Software. 

Ele é a evidência prática do meu processo de aprendizagem em controle de versão. Todo o histórico de commits, a gestão de branches e a resolução de conflitos documentada aqui demonstram a aplicação de boas práticas de engenharia de software.

## 📂 Índice e Estrutura do Repositório

Abaixo está o mapa de organização deste portfólio para facilitar a navegação e a avaliação:

* 📄 **`README.md`** — Você está aqui! Apresentação e índice do repositório.
* 🧠 **`REFLEXAO.md`** — Documento contendo minha reflexão pessoal sobre o processo de aprendizagem com Git (dificuldades, clarezas e dúvidas restantes).
* 📁 **`exercicios/`** — Diretório contendo as evidências práticas dos exercícios:
  * 📁 **`ex01-commits/`** — Arquivos utilizados para demonstrar o uso de commits semânticos.
  * 📁 **`ex02-branches/`** — Arquivos que receberam modificações em branches paralelas.
  * 📁 **`ex03-conflito/`** — Contém o arquivo `REGISTRO.md`, que documenta detalhadamente como um conflito de merge foi gerado e resolvido manualmente.
* 💻 **`projeto/`**
  * 📁 **`hello-world/`** — O mini-projeto base utilizado para exercitar as criações de features e commits ao longo da atividade.
